## Project Overview

**Duration**: June 2025 - November 2025  
**Affiliation**: Joint Project between Sichuan University and National University of Singapore  
**Role**: Core Team Member  
**Advisors**: Prof. Hongliang Guo, Prof. Marcelo Ang Jr., Prof. Jiancheng Lv, Prof. Yingjie Zhou

## Project Description

As a core member of the first lightweight autonomous golf cart project in Southwest China, I participated in full-stack system development from hardware modification to software architecture.

### Key Contributions

**Hardware System Integration**
- Completed wire-by-wire modification, electrification transformation, and multi-sensor integration based on a standard golf cart
- Built a reliable engineering-grade vehicle platform

**Software Architecture Development**
- Developed multi-source fusion localization and prediction algorithms to handle complex campus environments
- Built autonomous driving control system based on ROS1

**Real Vehicle Deployment and Testing**
- Completed open road testing at Sichuan University Jiang'an Campus
- Achieved autonomous cruising, turning, and dynamic obstacle avoidance capabilities

## Technology Stack

- **Framework**: ROS1, Arduino
- **Programming Languages**: C++, Python
- **Core Technologies**: SLAM, Multi-source Fusion Localization

## Achievements

- ✅ Successfully deployed the first lightweight autonomous golf cart in Southwest China universities
- ✅ Achieved autonomous navigation in real campus environments
- ✅ Established engineering foundation for future autonomous driving research
